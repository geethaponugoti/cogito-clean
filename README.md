# Luminary

Luminary is an AI-powered data analysis dashboard that enables users to upload datasets and automatically generate insights, visualizations, and business recommendations.

## 🚀 Features
- Upload CSV datasets
- Automated data profiling
- KPI metrics and summaries
- Interactive charts (bar, line, scatter)
- Correlation analysis
- Anomaly detection
- AI-style recommendations

## 🧠 How it works
1. User uploads dataset
2. System parses and extracts schema
3. Metrics and patterns are computed
4. Insights and charts are generated
5. Results are displayed in a dashboard

## 🛠️ Tech Stack
- React + TypeScript
- Vite
- Tailwind CSS
- Recharts (visualization)
- Supabase (backend)

## 📊 Use Cases
- Business data analysis
- Sales performance insights
- Customer segmentation
- Quick dataset exploration

## 🎯 Purpose
This project was built to demonstrate:
- Data analysis workflows
- Frontend dashboard design
- Visualization and UI engineering
- Full-stack integration with Supabase
